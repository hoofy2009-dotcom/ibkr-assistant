# 📊 主力行为分析指南

## 核心理念
**散户跟主力，主力跟趋势**。识别主力资金流向是散户生存的关键技能。

---

## 🎯 主力行为识别体系

### **1. 三大核心信号**

#### **💰 主力吸筹 (Accumulation)**
**特征**：
- 📉 **缩量下跌** → 主力静默买入，不引起散户注意
- 📈 **缩量上涨** → 筹码锁定，浮筹减少
- 🔄 **震荡洗盘** → 上下震荡，洗掉散户浮筹
- 💸 **大单托底** → 关键支撑位有大单接盘

**量价特征**：
```
价格: 低位横盘或阴跌
成交量: 持续萎缩(比前20日均量<70%)
换手率: 低(<3%)
形态: W底、圆弧底、箱体整理
```

**AI判断逻辑**：
```javascript
if (价格在低位 && 成交量萎缩 && 波动率降低) {
  volumeSignal = "accumulation" 💰🟢
  建议: "主力吸筹阶段,耐心等待突破,突破后跟进"
}
```

**实战案例**：
- NVDA 2023年3月: 从$210跌至$190横盘3周,成交量萎缩60%,主力吸筹,4月突破至$250(+32%)
- TSLA 2023年1月: 从$120底部横盘,成交量极度萎缩,主力建仓,随后涨至$210(+75%)

---

#### **⚠️ 主力出货 (Distribution)**
**特征**：
- 📈 **放量滞涨** → 成交量放大但涨幅微小(最危险)
- 🎢 **高位震荡** → 上下震荡吸引散户接盘
- 📉 **放量下跌** → 主力不计成本出逃
- 💥 **跳空低开** → 主力恐慌性抛售

**量价特征**：
```
价格: 高位或连续上涨后
成交量: 突然放大(比前20日均量>150%)
换手率: 高(>8%)
形态: M头、圆弧顶、头肩顶
```

**AI判断逻辑**：
```javascript
if (价格在高位 && 放量但涨幅<2% && 连续3天) {
  volumeSignal = "distribution" ⚠️🔴
  建议: "主力出货信号!立即减仓或清仓,不要幻想反弹"
  风险等级: 9/10
}
```

**实战案例**：
- ARKK 2021年2月: $150高位连续5天放量滞涨(日均成交量+200%),散户疯狂买入,随后3个月跌至$80(-47%)
- 某科技股2023年: 财报后涨5%但放量3倍,次日暴跌12%,典型的"利好出货"

---

#### **➡️ 资金中性 (Neutral)**
**特征**：
- 📊 成交量正常(90-110%均量)
- 💹 价格随大盘波动
- 🔀 无明显主力动作

**AI判断逻辑**：
```javascript
if (成交量正常 && 无明显异常) {
  volumeSignal = "neutral" ➡️
  建议: "主力无明显动作,关注技术信号和大盘走势"
}
```

## 🚨 主力陷阱识别 (散户必看)

### **陷阱1: 诱多拉升**
```
模式: 尾盘最后5分钟突然拉升3-5%
目的: 吸引散户次日追高接盘
识别: 成交量集中在尾盘,全天缩量
对策: 不追！次日低开概率80%
```

### **陷阱2: 砸盘洗筹**
```
模式: 开盘跳水-5%,然后缓慢回升
目的: 吓跑散户浮筹,主力低位回收
识别: 跳水时成交量小,回升时放量
对策: 确认是洗盘(有重大利好)才敢抄底
```

### **陷阱3: 对敲做量**
```
模式: 成交量暴增但价格不动
目的: 制造活跃假象,吸引散户
识别: 大单对倒,买卖盘相近
对策: 观望,等待真实突破
```

### **陷阱4: 财报出货**
```
模式: 财报超预期但当天或次日大跌
目的: 利用利好派发筹码
识别: 财报前连续上涨+财报日放巨量
对策: 财报前3天获利了结,不参与博弈
```

## 📈 主力行为量化指标

### **1. 成交量比率 (Volume Ratio)**
```javascript
VR = 当日成交量 / 过去20日平均成交量

VR < 0.5  = 极度萎缩 (可能吸筹)
VR 0.5-0.8 = 缩量
VR 0.8-1.2 = 正常
VR 1.2-2.0 = 放量
VR > 2.0   = 异常放量 (警惕出货)
```

### **2. 换手率 (Turnover Rate)**
```javascript
换手率 = 当日成交量 / 流通股本 × 100%

< 1%   = 冷门股,观望
1-3%   = 正常交易
3-7%   = 活跃,关注
7-15%  = 高度活跃,警惕主力动作
> 15%  = 异常,可能拉高出货
```

### **3. 量价背离 (Volume-Price Divergence)**
```javascript
// 顶背离 (卖出信号)
if (价格创新高 && 成交量萎缩) {
  信号: "顶背离,主力出货中"
  操作: 分批减仓
}

// 底背离 (买入信号)
if (价格创新低 && 成交量萎缩) {
  信号: "底背离,主力吸筹中"
  操作: 分批建仓
}
```

### **4. OBV能量潮 (On-Balance Volume)**
```javascript
// 简化计算
if (今日收盘 > 昨日收盘) {
  OBV += 今日成交量
} else {
  OBV -= 今日成交量
}

// 判断
if (价格上涨 && OBV上涨) → 主力买入,健康上涨
if (价格上涨 && OBV下跌) → 主力出货,警惕!
if (价格下跌 && OBV上涨) → 主力吸筹,逢低买入
```

## 🎯 实战应用策略

### **策略1: 跟随主力吸筹**
```
1. 等待股价在低位横盘(20日MA附近)
2. 观察成交量连续3天萎缩(<均量70%)
3. 出现首次放量突破(>均量120%)
4. 大盘环境配合(不能大跌)
5. 介入点: 突破日或回调确认日
6. 止损: -5% 或跌破吸筹区
```

### **策略2: 识别主力出货**
```
1. 警惕信号: 连续2天放量>150%但涨幅<2%
2. 确认信号: 第3天继续放量滞涨
3. 操作: 立即减仓50%,次日清仓
4. 不要: 等待反弹、加仓摊平、抱有幻想
```

### **策略3: 日内做T配合主力**
```
开盘: 观察集合竞价量能
  - 放量高开>3% → 可能冲高回落,先卖
  - 缩量低开<-2% → 可能低开高走,先买
  
盘中: 关注分时成交量
  - 分时量堆突破均线 → 跟进
  - 分时量堆但价不动 → 出货信号
  
尾盘: 警惕拉升陷阱
  - 尾盘5分钟拉升>3% → 不追,次日低开
```

## 💡 AI如何判断主力行为

### **智囊团 (advisor-v2.js)**
```javascript
// 1. 收集成交量数据(需要从Yahoo Finance或Finnhub获取)
const volume = stockData.volume
const avgVolume20 = stockData.avgVolume20days
const volumeRatio = volume / avgVolume20

// 2. 分析量价关系
const priceChange = (todayPrice - yesterdayPrice) / yesterdayPrice
const isUpday = priceChange > 0

// 3. 判断主力行为
if (volumeRatio < 0.7 && priceChange < 0.02 && inLowPosition) {
  return { 
    volumeSignal: "accumulation",
    advice: "主力吸筹阶段,耐心等待突破"
  }
}

if (volumeRatio > 1.5 && priceChange < 0.03 && inHighPosition) {
  return {
    volumeSignal: "distribution",
    advice: "⚠️ 主力出货!立即减仓",
    risk: 9
  }
}

// 4. 整合到AI分析
AI会收到: "成交量今日放大180%但涨幅仅2%,处于高位,判断主力出货"
AI输出: {
  action: "SELL",
  volumeSignal: "distribution",
  reason: "高位放量滞涨,主力出货信号明确"
}
```

## ⚠️ 散户警示

1. **不要盲目跟庄**: 主力也会亏损,散户要有自己的判断
2. **识别假信号**: 小盘股容易被操纵,要结合大盘环境
3. **止损纪律**: 判断错误时及时止损,不要死扛
4. **仓位管理**: 即使跟对主力,也要控制仓位(≤30%)

---

## 🎯 总结

**主力行为 = 成交量 + 价格 + 位置 + 时机**

- 💰 **吸筹**: 低位+缩量+横盘 → 耐心等突破
- ⚠️ **出货**: 高位+放量滞涨 → 立即撤离
- 🔄 **洗盘**: 中位+震荡+缩量 → 持股待涨
- 🚀 **拉升**: 突破+放量+趋势 → 跟进做多

**记住: 散户的优势是灵活,劣势是信息滞后。识别主力行为能提高胜率,但大盘趋势永远是第一位！**
